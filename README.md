# 🔐 Gerador de Senhas

Um gerador de senhas moderno e responsivo desenvolvido com **HTML**, **CSS** e **JavaScript**.

O projeto permite gerar senhas fortes e seguras de forma rápida, com opções personalizadas para aumentar a segurança das credenciais.

---

## 🚀 Funcionalidades

- ✅ Gerar senhas aleatórias
- ✅ Escolher tamanho da senha
- ✅ Incluir letras maiúsculas
- ✅ Incluir letras minúsculas
- ✅ Incluir números
- ✅ Incluir caracteres especiais
- ✅ Copiar senha para área de transferência
- ✅ Interface responsiva
- ✅ Armazenamento local com `localStorage`

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
